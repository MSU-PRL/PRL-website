Icon Support.

Any Tabs can be Accordions.

... and will be automatically when the responsive switch is achieved. The responsive switch can be anything from 200px to 20000px. Anything below the responsive switch will be a toggle. So if you want the tabs to always display as toggles / accordions, you should set to a high number like that. In the example on the right, you should see how easy it is to display an image accordion. The skin is skin-blue and the type is set to accordion.

Works on Black Themes.

This example has set skin-melbourne and tabs are on the left side. The transitions is set to slide. The background is set to black to show that ZoomTabs and Accordions looks fine in any enviroment.

Also, this example has enabled linking which means that any tab you click will be saved in the link for sharing or for remembering which tab you opened and remain opened.

FAQ Style.

This example is set to always be a toggle. It is build like a Frequently Asked Questions to show the capability of the plugin. The skin is skin-box and the transition is set to fade.

The accordion mode is set to toggle. An accordion leaves the other items closed while showing only the active one. A toggle lets the past items be open.
